Weather Data API

Enter dates with a dash - in between the 4 digit year, the 2 digit month and the 2 digit day

URL Format: http://127.0.0.1:5000/api/v1/station/date

URL Example: http://127.0.0.1:5000/api/v1/10/1988-10-25


Enter the station and year to see the station data for a single year

URL Year Format: http://127.0.0.1:5000/api/v1/yearly/station/year

URL Year Example: http://127.0.0.1:5000/api/v1/yearly/10/1988


Enter the station code for the station

URL Format: http://127.0.0.1:5000/api/v1/station

URL Station Example: http://127.0.0.1:5000/api/v1/10
